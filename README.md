# About the package

This package might be useful to create a user icon with their initials. Light weight and simple to use.


# Installation

`npm install adityask/userinitial --save`


# Usage

```
import {userinitial} from userinitial

const res = userinitial({
name: "Aditya Kadam",
height: "50px",
width: "50px",
root: "#root",
returnHTML: true
})


```
